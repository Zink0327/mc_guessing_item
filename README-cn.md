# Minecraft 猜物品

## 介绍

你能根据合成配方猜出隐藏的 Minecraft 物品吗？

## 游戏目标

有一个目标Minecraft物品，猜测区域会提供它的合成配方。你需要根据合成配方来猜出目标物品。

## 玩法指导

- 选择物品进行猜测，系统会显示从该物品合成目标物品需要多少步骤。
- 点击主猜测区域的1-9格以查看该格的合成配方。
- 所有物品一开始都是隐藏的，一旦猜中就会显示在对应的位置。
- 正确猜出目标物品即可获胜！

## 模糊匹配

- 系统可能会忽略颜色词（白色、橙色、红色等）、木材类型（橡木、云杉等）和材质词（染色、抛光、切割等）。
> 例如：输入“红羊毛”可能匹配“橙羊毛”，输入“红色玻璃板”可能匹配“紫色染色玻璃板”。
- 猜测记录会显示与答案距离小于3的正确物品，帮助你了解合成关系。
> 距离显示：距离表示该物品与答案相差多少合成步骤；“远”表示大于2步。

## 其他说明

- 有灰色背景的格子为空白格，不能点击。
- 物品信息截止到Minecraft JE 1.20.5。
- 游玩愉快！

## 如何启动

要启动游戏，首先需要在电脑上安装Node.js。然后在backend/目录下运行：

```
npm install
node server.js
```

然后在地址 http://localhost:4000/ 享受游戏吧！
